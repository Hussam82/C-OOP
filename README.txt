University Students database system
It reads a file then opens a menu which allows the user to enter his choice.
The user can :
-Enter '1' to enter student data
-Enter '2' to display all students data
-Enter '3' to modify student data
-Enter '4' to show student's grade
-Enter '5' to search for a student
-Enter '6' to delete a student
-Enter '7' to exit program
Then the file is updated when modifying on the database.
